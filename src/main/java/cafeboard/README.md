# 게시판(board)
## 게시판 생성
### method : PostMapping
### /boards
### http://localhost:8080/boards

## 게시판 목록 조회
### method : GetMapping
### /boards/{boardId}}
### http://localhost:8080/boards/1

```
{
    "boardName" : 게시판이름,
    [
        post1,
        post2,
        post3
    ]
}
```

## 게시판 수정
### method : PutMapping
### /boards/{boardId}
### http://localhost:8080/boards/1


## 게시판 삭제
### method : DeleteMapping
### /boards/{boardId}
### http://localhost:8080/boards/1
